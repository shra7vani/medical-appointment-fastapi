🏥 Medical Appointment System — FastAPI

A production-ready REST API built using FastAPI to manage doctors, appointments, and consultation workflows. This project demonstrates backend development best practices including API design, validation, and structured workflows.

🛠️ Tech Stack

Python

FastAPI

Pydantic

Swagger UI (OpenAPI)

🚀 Key Features

Doctor Management — Create, update, view, and delete doctor records

Appointment System — Book and manage patient appointments

Workflow Handling — Structured lifecycle:
👉 Book → Confirm → Complete

Search & Filtering — Easily find doctors and appointments

Sorting & Pagination — Efficient data handling for large datasets

Validation Rules — Prevent duplicate entries and enforce constraints

Authentication Ready — Designed to integrate JWT-based authentication

Interactive API Docs — Auto-generated Swagger UI

⚙️ Running the Project

Install dependencies

Start the FastAPI server

Access the API locally

📖 API Documentation

Once the server is running, open:
👉 http://127.0.0.1:8000/docs

This provides an interactive interface to test all endpoints.
